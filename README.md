This blog is based on Hyde which is a brazen two-column [Jekyll](http://jekyllrb.com) theme that pairs a prominent sidebar with uncomplicated content. It's based on [Poole](http://getpoole.com), the Jekyll butler.

## Development

Use a Ruby selected outside this repo. The macOS system Ruby is too old for this bundle.

With an installed `mise` Ruby:

```sh
make install
make build
make serve
```

If your shell already points to a compatible Ruby, skip the default `mise` runner:

```sh
make install RUBY_RUNNER=
make build RUBY_RUNNER=
make serve RUBY_RUNNER=
```
